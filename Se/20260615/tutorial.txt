2. Brefly explain how the project time plan (gantt chart) can used in managing the projects time ?
A gantt chart is a visual plan for the project that shows each element of the project and time that need to be finished.
It can be used to manage the project time by showing the start and end dates of each task, and a manager can know the project is going on time and 
each task that are allocated to developers going to be completed on time. It can also be used to identify any delays in the project and take corrective actions.
